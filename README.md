# TEMPORARY workshop data

These CSVs are a subsampled, minimal-column extract of licensed GLSS6/DHS
teaching data, hosted ONLY to drive the interactive WebR demo during the
IndabaX Ghana 2026 workshop.

## Host these for the workshop window only
1. Create a separate PUBLIC repo, e.g. `acet-workshop-data-temp`.
2. Put these four CSVs in it.
3. The WebR .qmd fetches them at:
   https://raw.githubusercontent.com/<user>/acet-workshop-data-temp/main/webr_glss_train.csv  (etc.)
